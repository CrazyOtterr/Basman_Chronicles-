ШАГ 1. Открыть сцену
────────────────────
• В Unity откройте сцену: Assets → Scenes → Level0 → ColleagueCabinet
• Или дважды кликните на ColleagueCabinet.unity в Project

ШАГ 2. Добавить окно головоломки
────────────────────────────────
• В окне Project найдите папку Assets/Prefabs
• Найдите префаб "Puzzle" (тот же, что у карты)
• Перетащите "Puzzle" в окно Hierarchy
• Опустите его ВНУТРЬ Canvas (под ===UI=== → Canvas)
  — Должен стать дочерним объектом Canvas, рядом с Puzzle и ExitButton

• Кликните на новый объект Puzzle в Hierarchy
• В Inspector сверху переименуйте: вместо "Puzzle" впишите "PuzzlePipe"
  (клик по имени вверху Inspector → ввести новое имя)

ШАГ 3. Добавить куски труб внутрь окна
──────────────────────────────────────
• В Hierarchy РАЗВЕРНИТЕ PuzzlePipe (клик по стрелке слева)
• Увидите: PuzzleBackground, PuzzlePanel
• Разверните PuzzlePanel
• В Project найдите префаб "PipePuzzle" (Assets/Prefabs/PipePuzzle)
• Перетащите PipePuzzle ВНУТРЬ PuzzlePanel
  — PipePuzzle должен стать дочерним объектом PuzzlePanel

ШАГ 4. Выключить окно по умолчанию
──────────────────────────────────
• Выберите PuzzlePipe в Hierarchy
• В Inspector найдите галочку рядом с именем "PuzzlePipe" вверху
• СНИМИТЕ галочку (активность)
  — Объект станет серым в Hierarchy, окно не будет видно при старте

ШАГ 5. Создать кнопку для теста
────────────────────────────────
• В Hierarchy кликните ПРАВОЙ кнопкой по ColleagueCabinet (корень сцены)
• Выберите: Create Empty
• Появится пустой объект — переименуйте его в "PipeTestOpener"
• Перетащите PipeTestOpener в секцию ===Managers=== (для порядка)

ШАГ 6. Настроить открытие по клавише P
──────────────────────────────────────
• Выберите PipeTestOpener в Hierarchy
• В Inspector нажмите "Add Component"
• В поиске введите: PipePuzzleTestOpener
• Выберите компонент PipePuzzleTestOpener

• В компоненте появится поле "Pipe Puzzle Window"
• Перетащите туда объект PuzzlePipe из Hierarchy
  (просто перетащите PuzzlePipe в это поле)

ШАГ 7. Сохранить и запустить
─────────────────────────────
• File → Save (или Ctrl+S)
• Нажмите кнопку Play ▶ вверху Unity
• В игре нажмите клавишу P на клавиатуре
• Должно открыться окно с трубами
• Кнопка X закрывает окно
• Куски труб можно перетаскивать мышью

───────────────────────────────────────────────────────────────
Если спрайты труб не видны (белые квадраты):
• Откройте префаб PipePuzzle (двойной клик в Project)
• Для каждого куска (pipe_straight_h, pipe_corner и т.д.):
  — Выберите объект
  — В Image → Source Image перетащите спрайт из Assets/Sprites/PipePuzzle/
