Подсветка интерактивных предметов при наведении (Ivan Stalmahov)
================================================================

Скрипт: PnC_InteractiveHighlightManager.cs

Что делает:
  При запуске сцены находит все объекты с компонентом PnC_InteractiveItem,
  создаёт для каждого белый полупрозрачный дубликат спрайта и показывает его
  при наведении курсора мыши. Сам PnC_InteractiveItem не изменён.

Как использовать:
  1. Открой любую сцену, где есть интерактивные предметы (например
     Scenes/Level0/ColleagueCabinet или Scenes/QuestSceneMainPuzzleMap).
  2. Создай пустой GameObject (ПКМ в Hierarchy → Create Empty).
  3. Добавь на него компонент: Add Component → Pn C Interactive Highlight Manager.
  4. Запусти сцену — при наведении на объекты с PnC_InteractiveItem появится
     белая подсветка.

Настройки в Inspector:
  - Highlight Alpha — прозрачность подсветки (0.1–0.9).
  - Sorting Order Offset — на сколько выше порядок отрисовки подсветки (по умолчанию 1).

Тестовая сцена:
  Можно использовать любую существующую сцену с интерактивными объектами;
  при необходимости скопируй сцену в эту папку и добавь на неё пустой объект
  с PnC_InteractiveHighlightManager.
